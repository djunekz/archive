# Archive - Termux Tools Installer

Archive is a collection of tools for Termux that makes it easy to install popular apps and scripts.

## Features

- Quickly install various Termux tools.
- Direct integration with [Termux App Store](https://github.com/djunekz/termux-app-store).
- Lightweight, user-friendly, suitable for both beginners and advanced users.

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/djunekz/archive
cd archive
```
